
# BMP Image Processor

Статус: 🟩Завершён🟩

Данный проект - курсовая работа за второй семестр, посвещённая обработке изображений формата BMP. Программа написана на языке C в виде статической библиотеки для работы с графикой и консольной утилиты на её основе. Программ реализует чтение, изменение и запись изображений без использования сторонних библиотек.

# Функционал

- поддержка только 24-битных не сжатых изображений
- проект разделён на графическую часть (`libbmp.a`) и CLI-интерфейс (`bmp-editor`)
- полная документация кода, сгенерированная при помощи Doxygen
- изменение изображений
	- рисование квадрата с диагоналями
	- наложение RGB-фильтра на изображение
	- поворот части изображения на 90/180/270 градусов
	- отражение части изображение по оси X или Y
	- отражение части изображения по диагонали
	- рисование равностороннего ромба

# Сборка и запуск

Для сборки потребуется компилятор gcc.

```bash
# Сборка в статическую библиотеку libbmp.a и компиляция исполняемого файла bmp-editor
make

# Сборка с включенным флагом AddressSanitizer для отладки памяти
make sanitize=1

# Запуск с выводом справки и информации о тестовой картинке
./bmp-editor -i -h ./images/sample.bmp
```

# Генерация документации

В корне проекта находится файл конфигурации `Doxyfile`, позволяющий сгенерировать документацию. Для создания локального сайта с документацией необходимо выполнить команду:
```bash
doxygen Doxyfile
```
после чего в папке `html/` будет собрана документация. Увидеть её можно, открыв файл `html/index.html` в любом браузере.

# Структура проекта

- `src/` - исходный код утилиты и модулей библиотеки
- `include/`  - заголовочные файлы с подробной Doxygen-документацией
- `images/` - тестовые изображения
- `obj/` - временная папка для объектных файлов, создаваемая при сборке
- `Doxyfile` - файл конфигурации для генератора документации Doxygen
- `.clang-format` - файл конфигурации стиля кода
- `Makefile` - файл контроля сборки программы (поддерживает сборку исполняемого файла, сборку библиотеки и санитайзер)
