
# BMP Image Processor

Данный проект - доделанная и доведённая до ума курсовая работа за второй семестр, посвященная обработке изображений формата BMP. Программа написана на языке C и позволяет производить чтение, изменение и запись изображений без использования сторонних библиотек.

# Функционал

- поддержка только 24-битных не сжатых изображений
- изменение изображений
	- рисование квадрата с диагоналями
	- наложение RGB-фильтра на изображение
	- поворот части изображения на 90/180/270 градусов
	- отражение части изображение по оси X или Y
	- отражение части изображения по диагонали

# Сборка и запуск

Для сборки потребуется компилятор gcc.

```bash
# Комплияция в исполняемый файл bmp-editor
make
# Запуск с тестовой картинкой
./bmp-editor -i -h ./images/sample.bmp
```

# Структура проекта

- `src/` - исходный код
- `include/`  - заголовочные файлы
- `images/` - тестовые изображения
- `.clang-format` - файл конфигурации стиля кода
- `Makefile` - файл контроля сборки программы
